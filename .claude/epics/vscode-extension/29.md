# Task: Implement "CCPM: Complete Current Task" command

## Description
Create a VS Code command that marks the currently active task as completed. This command should find the task with "in_progress" status and transition it to "completed" with appropriate metadata updates.

## Acceptance Criteria
- [ ] Command "CCPM: Complete Current Task" appears in command palette
- [ ] Command identifies the task currently in progress
- [ ] Updates task status from "in_progress" to "completed"
- [ ] Updates task metadata (updated timestamp)
- [ ] Provides user feedback on which task was completed
- [ ] Handles edge cases (no task in progress, multiple tasks in progress)

## Technical Details
- Implement command registration in extension.ts
- Parse current epic's task files
- Find task(s) with status "in_progress"
- Update task frontmatter status field to "completed"
- Update timestamp in frontmatter
- Write updated task file back to disk
- Show notification or output message to user
- Consider integration with GitHub API to close issue if synced
- Handle case where multiple tasks are in progress (prompt user to select)

## Dependencies
- [ ] Task file structure is defined
- [ ] Frontmatter parsing and writing capabilities
- [ ] Current epic detection logic
