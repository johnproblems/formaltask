# Task: Implement notification manager service

## Description
Create a centralized notification manager service that handles displaying notifications to users within the VS Code extension. This service will manage notification queuing, display logic, and provide a consistent interface for triggering notifications across the extension.

## Acceptance Criteria
- [ ] NotificationManager class created with singleton pattern
- [ ] Support for info, warning, and error notification types
- [ ] Notification queue system implemented
- [ ] Methods for showing, dismissing, and managing notifications
- [ ] Integration with VS Code's notification API

## Technical Details
- Implement as a singleton service in `src/services/NotificationManager.ts`
- Use VS Code's `window.showInformationMessage`, `window.showWarningMessage`, and `window.showErrorMessage` APIs
- Include notification queue to prevent overwhelming users
- Support action buttons in notifications where needed
- Add configuration options for notification preferences

## Dependencies
- [ ] VS Code extension API
- [ ] TypeScript type definitions for VS Code
