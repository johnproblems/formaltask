# Task: Create Quick Pick menu with actions

## Description
Create a Quick Pick menu that appears when the status bar item is clicked, providing quick access to task-related actions such as viewing task details, changing task status, opening task file, and listing all tasks.

## Acceptance Criteria
- [ ] Quick Pick menu opens when status bar item is clicked
- [ ] Menu displays relevant task actions (view, edit, change status, list tasks)
- [ ] Each menu item triggers the appropriate action
- [ ] Menu items show appropriate icons and descriptions

## Technical Details
- Use `vscode.window.showQuickPick()` API
- Create QuickPickItem objects with labels, descriptions, and icons
- Register command handlers for each menu action
- Implement actions: view task details, open task file, change status, list all tasks
- Handle menu cancellation gracefully
- Consider adding keyboard shortcuts for common actions
