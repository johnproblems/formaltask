# Task: Add click handler to open task files in editor

## Description
Implement click handlers for task and epic nodes in the tree view that open the corresponding markdown files in the VS Code editor. This allows users to quickly navigate to and edit task/epic files directly from the tree view.

## Acceptance Criteria
- [ ] Clicking a task node opens the task markdown file in the editor
- [ ] Clicking an epic node opens the epic.md file in the editor
- [ ] File opens in the active editor group
- [ ] File is focused and brought to front if already open
- [ ] Error handling for missing or inaccessible files
- [ ] Preserves cursor position when reopening files

## Technical Details
- Set `command` property on TreeItem objects:
  ```typescript
  command: {
    command: 'vscode.open',
    title: 'Open File',
    arguments: [vscode.Uri.file(filePath)]
  }
  ```
- For TaskNode: point to task markdown file path
- For EpicNode: point to epic.md file path
- File paths should be absolute paths resolved during node creation
- Use built-in `vscode.open` command (no custom command needed)
- Alternative: implement custom command for additional logic (preview mode, etc.)
- File location: Update `src/treeview/nodes/TaskNode.ts` and `src/treeview/nodes/EpicNode.ts`
