# Task: Implement "CCPM: Sync Progress" command

## Description
Create a VS Code command that synchronizes local task/epic progress with GitHub. This command should push local changes to GitHub issues and pull updates from GitHub to local task files.

## Acceptance Criteria
- [ ] Command "CCPM: Sync Progress" appears in command palette
- [ ] Command pushes local task updates to GitHub issues
- [ ] Command pulls GitHub issue updates to local task files
- [ ] Handles authentication with GitHub API
- [ ] Provides progress feedback during sync
- [ ] Reports sync results (success/failures) to user
- [ ] Handles conflicts gracefully

## Technical Details
- Implement command registration in extension.ts
- Integrate with GitHub API (using Octokit or similar)
- Read local task files and extract github issue numbers
- Update GitHub issues based on local task status changes
- Fetch GitHub issue updates and update local task files
- Handle authentication (use VS Code secrets API or GitHub token)
- Show progress indicator during sync operation
- Display sync results in output channel or notification
- Implement conflict resolution strategy (local wins, remote wins, or prompt)

## Dependencies
- [ ] GitHub API integration library
- [ ] Authentication mechanism for GitHub
- [ ] Task file structure with github field defined
