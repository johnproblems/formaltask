# Task: Implement progress percentage display for epics

## Description
Add progress percentage calculations and display for epic nodes in the tree view. Show completion percentage based on the status of child tasks, providing a quick overview of epic completion status.

## Acceptance Criteria
- [ ] Progress percentage calculated as (done + skipped) / total tasks
- [ ] Percentage displayed in epic node label (e.g., "Epic Name (75%)")
- [ ] Percentage updates when task statuses change
- [ ] Shows 0% for epics with no tasks
- [ ] Shows 100% when all tasks are done or skipped
- [ ] Progress calculation excludes tasks marked as not counting toward progress (if applicable)

## Technical Details
- Implement progress calculation in `src/treeview/progressCalculator.ts`
- Formula: `(done_count + skipped_count) / total_tasks * 100`
- Round to nearest integer for display
- Update EpicNode.label to include percentage:
  - Format: `${epicName} (${progress}%)`
  - Example: "VS Code Extension (60%)"
- Add progress property to EpicNode for use in other features
- Consider adding progress bar in tooltip
- Handle edge cases:
  - No tasks: show "(0%)" or "(-)"
  - All skipped: show "(100%)"
