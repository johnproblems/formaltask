# Task: Initialize extension project with TypeScript + webpack config

## Description
Set up the foundational project structure for the VSCode extension with TypeScript compilation and webpack bundling. This includes creating the basic directory structure, configuring TypeScript with strict mode, and setting up webpack for efficient bundling of the extension code.

## Acceptance Criteria
- [ ] Project directory structure created (src/, out/, dist/, test/)
- [ ] TypeScript configuration (tsconfig.json) with strict mode and VSCode types

- [ ] Source maps enabled for debugging
- [ ] Build scripts functional and producing bundled output

- Set up separate webpack configs for development and production
- Configure output to `dist/` directory
- Include VSCode extension types (@types/vscode)
- Enable source map generation for debugging
