# Task: Create "Sync to GitHub" button with terminal integration

## Description
Add a "Sync to GitHub" button to the progress panel that triggers git operations in an integrated terminal. The button should commit changes to progress.md and push to the remote repository, providing visual feedback on the operation status.

## Acceptance Criteria
- [ ] "Sync to GitHub" button added to panel UI
- [ ] Button opens integrated terminal on click
- [ ] Terminal executes git add, commit, push commands
- [ ] Commit message auto-generated with timestamp
- [ ] Success/error feedback shown in panel
- [ ] Button disabled during sync operation
- [ ] Git status checked before sync attempt

## Technical Details
- Add sync button to webview HTML
- Implement message handler for sync button clicks
- Use vscode.window.createTerminal() for git operations
- Execute commands: `git add progress.md`, `git commit -m "Update progress notes [timestamp]"`, `git push`
- Check git status with `git status --porcelain` before sync
- Show notification on success/failure
- Implement loading spinner during operation
- Files affected: `src/commands/syncProgress.ts` (new), `src/webview/progressPanelProvider.ts`, webview HTML/script

## Dependencies
- [ ] Task 20: Create webview panel provider for progress notes
- [ ] Git installed and configured in workspace
- [ ] GitHub remote configured
