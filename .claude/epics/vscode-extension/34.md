# Task: Create status bar item with current task info

## Description
Create a VS Code status bar item that displays the current task information. The status bar item should be visible in the bottom status bar and show relevant task details such as task number, name, and status.

## Acceptance Criteria
- [ ] Status bar item is created and visible in VS Code status bar
- [ ] Status bar item displays current task number and name
- [ ] Status bar item shows appropriate icon based on task status
- [ ] Status bar item is clickable and interactive

## Technical Details
- Use VS Code's `vscode.window.createStatusBarItem()` API
- Position the status bar item appropriately (e.g., StatusBarAlignment.Left)
- Set appropriate priority to control placement relative to other items
- Add command association for click handling
- Implement proper disposal/cleanup when extension deactivates
