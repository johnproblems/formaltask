# Task: Implement task selection handler (tree → panel sync)

## Description
Implement bidirectional synchronization between the task tree view and the progress notes panel. When a user selects a task in the tree view, the panel should automatically update to show that task's progress notes and scroll to the relevant section.

## Acceptance Criteria
- [ ] Tree view selection triggers panel update
- [ ] Panel loads corresponding task's progress notes
- [ ] Panel auto-scrolls to selected task section
- [ ] Selection state maintained across panel refreshes
- [ ] Loading state shown during content fetch
- [ ] Error handling for missing/invalid task data

## Technical Details
- Implement event listener for tree view selection changes
- Create message protocol between extension and webview
- Implement scrollIntoView with smooth scrolling for task sections
- Cache task-to-section mapping for performance
- Handle race conditions with debouncing/throttling
- Store selection state in extension context
- Files affected: `src/treeView/taskTreeView.ts`, `src/webview/progressPanelProvider.ts`, webview script
