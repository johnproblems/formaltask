# Task: Add GitHub token SecretStorage integration (encrypted)

## Description
Implement secure storage for GitHub authentication tokens using VS Code's SecretStorage API. This ensures sensitive credentials are encrypted and stored securely rather than in plain-text configuration files.

## Acceptance Criteria
- [ ] SecretStorage integration implemented for GitHub tokens
- [ ] Token storage and retrieval methods created
- [ ] Secure token deletion/clearing functionality
- [ ] Migration path for any existing plain-text tokens
- [ ] Token validation before storage
- [ ] User prompts for token input with proper UI

## Technical Details
- Create `src/services/SecretManager.ts`
- Use `ExtensionContext.secrets` API for encrypted storage
- Implement methods:
  - `storeGitHubToken(token: string): Promise<void>`
  - `getGitHubToken(): Promise<string | undefined>`
  - `deleteGitHubToken(): Promise<void>`
  - `hasGitHubToken(): Promise<boolean>`
- Add token format validation (GitHub PAT format)
- Implement input box UI for token entry
- Add "Set GitHub Token" command to command palette
- Handle token refresh/update scenarios
- Clear sensitive data from memory after use
- Add proper error handling for storage failures
