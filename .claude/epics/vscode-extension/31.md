# Task: Implement "CCPM: Refresh All" command

## Description
Create a VS Code command that refreshes all CCPM-related views and data in the extension. This command should reload task files, update tree views, and refresh any cached data to ensure the UI reflects the current state.

## Acceptance Criteria
- [ ] Command "CCPM: Refresh All" appears in command palette
- [ ] Command reloads all task and epic files from disk
- [ ] Refreshes all tree views and UI components
- [ ] Clears any cached data
- [ ] Provides user feedback when refresh is complete
- [ ] Handles errors gracefully (missing files, parse errors)

## Technical Details
- Implement command registration in extension.ts
- Create refresh logic for all data providers
- Re-read task and epic files from .claude/epics directory
- Trigger tree view refresh events
- Clear any in-memory caches
- Re-parse all markdown files
- Update UI components with fresh data
- Show notification when refresh completes
- Log any errors encountered during refresh
