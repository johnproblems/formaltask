# Task: Implement context menu command handlers (7 actions)

## Description
Implement the command handlers for all 7 context menu actions that users can perform on epic and task items. Each handler should process the selected item, perform the appropriate operation, and update the tree view to reflect changes.

## Acceptance Criteria
- [ ] All 7 command handlers are implemented and functional
- [ ] Handlers properly receive selected tree view items as parameters
- [ ] Each action performs the correct operation on the underlying data
- [ ] Tree view is refreshed after state-changing operations
- [ ] Error handling is implemented for all commands
- [ ] User feedback is provided via notifications or status messages

## Technical Details
- Implement handlers for these 7 actions:
  1. **Start/Stop**: Toggle epic or task active state
  2. **Mark Blocked**: Set task status to blocked
  3. **Set Priority**: Prompt user to select priority level
  4. **Add Dependency**: Show picker to select dependent task
  5. **View Dependencies**: Display dependency graph or list
  6. **Edit**: Open task/epic file for editing
  7. **Delete**: Remove task/epic with confirmation prompt
- Register command handlers using `vscode.commands.registerCommand`
- Use VS Code APIs for user input (QuickPick, InputBox)
- Update underlying task/epic data files
- Trigger tree view refresh after modifications
- Handle edge cases (e.g., circular dependencies, invalid states)
