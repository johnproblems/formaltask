# Task: Register context menu contributions in package.json

## Description
Register all context menu contributions in the VS Code extension's package.json file to enable right-click context menu actions for epic and task items in the tree view. This includes defining menu items, their visibility conditions, and linking them to command identifiers.

## Acceptance Criteria
- [ ] Context menu contributions are properly registered in package.json
- [ ] Menu items are defined for both epic and task contexts
- [ ] All 7 action commands are registered with appropriate icons and labels
- [ ] Menu visibility conditions are correctly specified using when clauses
- [ ] Command identifiers follow VS Code naming conventions

## Technical Details
- Register menus under the `contributes.menus` section in package.json
- Define context menu items for the custom view (e.g., `view/item/context`)
- Configure 7 context menu actions:
  - Start/Stop epic or task
  - Mark task as blocked
  - Set task priority
  - Add dependency
  - View dependencies
  - Edit task/epic
  - Delete task/epic
- Use `when` clauses to control menu item visibility based on view context
- Link each menu item to its corresponding command ID
- Add appropriate icons using $(iconName) syntax

## Dependencies
- [ ] VS Code extension base structure exists
- [ ] Command IDs are defined or will be defined in command handlers
