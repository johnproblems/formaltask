# Task: Implement "CCPM: Start Next Task" command

## Description
Create a VS Code command that automatically starts the next available task in the current epic. This command should identify the highest priority open task and transition it to "in_progress" status.

## Acceptance Criteria
- [ ] Command "CCPM: Start Next Task" appears in command palette
- [ ] Command identifies the next open task based on priority/order
- [ ] Updates task status from "open" to "in_progress"
- [ ] Updates task metadata (updated timestamp)
- [ ] Provides user feedback on which task was started
- [ ] Handles edge cases (no open tasks, task already in progress)

## Technical Details
- Implement command registration in extension.ts
- Parse current epic's task files
- Identify next task to start (based on task number or dependencies)
- Update task frontmatter status field
- Update timestamp in frontmatter
- Write updated task file back to disk
- Show notification or output message to user
- Consider integration with GitHub API if task is synced

## Dependencies
- [ ] Task file structure is defined
- [ ] Frontmatter parsing and writing capabilities
- [ ] Current epic detection logic
