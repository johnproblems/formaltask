# Task: Implement "CCPM: Add Task" command

## Description
Create a VS Code command that opens an integrated terminal and runs the CCPM CLI command to add a new task. This provides a convenient way to create tasks directly from the VS Code interface.

## Acceptance Criteria
- [ ] Command "CCPM: Add Task" appears in command palette
- [ ] Command opens VS Code integrated terminal
- [ ] Terminal executes "ccpm task add" command
- [ ] User can interact with the CLI prompts in the terminal
- [ ] Terminal remains open after command completes

## Technical Details
- Implement command registration in extension.ts
- Use VS Code terminal API to create/show terminal
- Execute "ccpm task add" command in the terminal
- Ensure terminal is focused for user interaction
- Handle cases where ccpm CLI is not installed
- Consider terminal reuse vs creating new terminal each time

## Dependencies
- [ ] CCPM CLI is installed and available in PATH
- [ ] VS Code terminal API integration
