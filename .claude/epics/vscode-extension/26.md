# Task: Implement "CCPM: Show Epic Status" command

## Description
Create a VS Code command that displays the current status of all epics in the project. This command should parse epic files and present a comprehensive overview of epic progress, task counts, and overall health.

## Acceptance Criteria
- [ ] Command "CCPM: Show Epic Status" appears in command palette
- [ ] Command displays all epics with their current status
- [ ] Shows task count (completed/total) for each epic
- [ ] Displays epic metadata (created, updated, github link if available)
- [ ] Output is formatted and easy to read

## Technical Details
- Implement command registration in extension.ts
- Create epic parser to read .claude/epics/*/*.md files
- Parse frontmatter to extract epic metadata
- Count tasks by status (open, in_progress, completed)
- Display results in VS Code output channel or webview
- Handle cases where epic directory doesn't exist

## Dependencies
- [ ] Epic file structure is defined
- [ ] Frontmatter parsing library available
