# Task: Implement "CCPM: Copy Issue Number" command

## Description
Create a VS Code command that copies the GitHub issue number of the current task to the clipboard. This provides a convenient way to reference issues in commit messages, comments, or other documentation.

## Acceptance Criteria
- [ ] Command "CCPM: Copy Issue Number" appears in command palette
- [ ] Command detects the currently active/selected task
- [ ] Extracts GitHub issue number from task metadata
- [ ] Copies issue number to clipboard
- [ ] Provides user feedback (notification or status bar message)
- [ ] Handles edge cases (no GitHub link, invalid format)

## Technical Details
- Implement command registration in extension.ts
- Detect current task from active editor or tree view selection
- Parse task frontmatter to extract github field
- Extract issue number from github field (handle both numbers and URLs)
- Use VS Code clipboard API (env.clipboard.writeText) to copy to clipboard
- Show notification confirming copy action
- Handle missing github field gracefully (show error message)
- Consider context menu integration for tree view items
- Support different formats (#123, 123, or full URL)

## Dependencies
- [ ] Task file structure with github field
- [ ] VS Code clipboard API access
