# Task: Implement tooltip markdown generation (7 required fields)

## Description
Create a markdown generator function that formats task information into a rich tooltip display. The tooltip must include 7 required fields: task name, status, created date, updated date, dependencies, effort estimate, and description. The markdown should be well-formatted and visually appealing.

## Acceptance Criteria
- [ ] Markdown generation function created
- [ ] All 7 required fields are included in the tooltip
- [ ] Markdown formatting is clean and readable
- [ ] Function handles missing or optional field data gracefully
- [ ] Tooltip renders correctly in VSCode hover popup

## Technical Details
- Create a utility function that accepts task data and returns formatted markdown string
- Required fields to display:
  1. Task name/title
  2. Status (open/in-progress/completed)
  3. Created date
  4. Updated date
  5. Dependencies (depends_on)
  6. Effort estimate (size and hours)
  7. Description
- Use markdown formatting: headers, bold, bullet points, code blocks as appropriate
- Handle edge cases (null values, empty arrays, etc.)
- File location: Create in `src/utils/` or integrate into HoverProvider

- [ ] VSCode markdown rendering support
