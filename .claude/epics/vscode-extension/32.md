# Task: Implement "CCPM: View on GitHub" command

## Description
Create a VS Code command that opens the GitHub issue associated with the current task in the user's default browser. This provides quick access to view task details, comments, and updates on GitHub.

## Acceptance Criteria
- [ ] Command "CCPM: View on GitHub" appears in command palette
- [ ] Command detects the currently active/selected task
- [ ] Extracts GitHub issue URL from task metadata
- [ ] Opens URL in default browser
- [ ] Handles edge cases (no GitHub link, invalid URL)
- [ ] Works from both tree view and active editor

## Technical Details
- Implement command registration in extension.ts
- Detect current task from active editor or tree view selection
- Parse task frontmatter to extract github field
- Construct GitHub issue URL from issue number and repo
- Use VS Code env.openExternal API to open URL in browser
- Handle missing github field gracefully (show error message)
- Support both issue numbers and full URLs in github field
- Consider context menu integration for tree view items

## Dependencies
- [ ] Task file structure with github field
- [ ] Repository URL configuration or detection
