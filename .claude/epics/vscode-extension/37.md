# Task: Add status bar update on task selection changes

## Description
Implement automatic updates to the status bar item when the current task selection changes. This includes watching for task file changes, status updates, and user actions that change the active task.

## Acceptance Criteria
- [ ] Status bar updates automatically when active task changes
- [ ] Status bar reflects task status changes in real-time
- [ ] File system watcher detects task file modifications
- [ ] Status bar updates when task is created, modified, or deleted

## Technical Details
- Implement FileSystemWatcher for task directory monitoring
- Create event handlers for task change events
- Subscribe to relevant VS Code workspace events
- Implement debouncing to prevent excessive updates
- Update status bar text using formatting logic from Task 35
- Ensure proper disposal of watchers and event subscriptions
- Handle cases where no task is selected (show default state)

- [ ] Task 35: Status text formatting logic implemented
- [ ] Task file structure and location defined
