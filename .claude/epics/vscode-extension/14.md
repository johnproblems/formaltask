# Task: Create TreeItem classes for Epic and Task nodes

## Description
Implement custom TreeItem classes (EpicNode and TaskNode) that extend vscode.TreeItem to represent epics and tasks in the tree view. These classes will encapsulate the display logic, icons, tooltips, and metadata for each node type.

## Acceptance Criteria
- [ ] EpicNode class extends vscode.TreeItem with epic-specific properties
- [ ] TaskNode class extends vscode.TreeItem with task-specific properties
- [ ] Both classes include collapsible state configuration
- [ ] Classes store metadata (file path, status, frontmatter data)
- [ ] Appropriate context values set for command targeting
- [ ] Tooltip shows relevant information (status, dates, description preview)

## Technical Details
- Create `src/treeview/nodes/EpicNode.ts` and `src/treeview/nodes/TaskNode.ts`
- EpicNode properties:
  - label: epic name from frontmatter
  - collapsibleState: Expanded or Collapsed
  - contextValue: 'epic' for command filtering
  - iconPath: folder icon or custom epic icon
- TaskNode properties:
  - label: task name from frontmatter
  - collapsibleState: None
  - contextValue: 'task' for command filtering
  - command: open file on click
- Store file path and parsed frontmatter for later use

## Dependencies

- [ ] TreeDataProvider (Task 13) - for integration
- [ ] Frontmatter parser utilities
